# 安装说明

本应用为内部使用版本，未经过代码签名。首次安装时需要按照以下步骤操作。

## macOS 安装

### 1. 安装应用

1. 双击 `.dmg` 文件打开磁盘映像
2. 将应用图标拖动到 `Applications` 文件夹

### 2. 首次打开

由于应用未经过 Apple 公证，首次打开时会被 macOS 阻止。请按照以下步骤操作：

**方法一：右键打开（推荐）**

1. 打开 `Applications` 文件夹
2. 找到应用图标
3. **按住 Control 键点击**应用图标（或右键点击）
4. 选择"打开"
5. 在弹出的对话框中点击"打开"

**方法二：系统设置**

1. 尝试双击打开应用（会被阻止）
2. 打开"系统设置" > "隐私与安全性"
3. 在"安全性"部分，点击"仍要打开"
4. 在确认对话框中点击"打开"

### 3. 后续使用

完成首次打开后，之后可以正常双击打开应用。

## Windows 安装

### 1. 运行安装程序

双击 `.exe` 安装程序。

### 2. 绕过 SmartScreen 警告

如果出现 Windows Defender SmartScreen 警告：

1. 点击"更多信息"
2. 点击"仍要运行"
3. 按照安装向导完成安装

### 3. 后续使用

安装完成后，可以从开始菜单或桌面快捷方式启动应用。

## 常见问题

### Q: 为什么会出现安全警告？

A: 本应用为内部使用版本，未经过代码签名和公证。这是正常现象，不影响使用。

### Q: 应用安全吗？

A: 是的。应用来源可信，只是未经过官方签名流程。

### Q: 每次打开都需要右键吗？

A: 不需要。只有首次打开时需要，之后可以正常双击打开。

### Q: 如何卸载？

**macOS**:
1. 打开 `Applications` 文件夹
2. 将应用拖到废纸篓

**Windows**:
1. 打开"设置" > "应用"
2. 找到应用并点击"卸载"
